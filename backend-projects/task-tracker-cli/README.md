# Task Tracker CLI 📝

A simple and lightweight Command Line Interface (CLI) application to track and manage your tasks. Built entirely with Vanilla Node.js (No external libraries).

This project is part of the [Roadmap.sh Backend Developer Projects](https://roadmap.sh/projects/task-tracker).

## ✨ Features

- **Add** a new task (Auto-generates custom ID like `task-001`).
- **Update** an existing task's description.
- **Delete** a task.
- **Mark** a task status as `in-progress` or `done`.
- **List** all tasks or filter them by status (`todo`, `in-progress`, `done`).
- **Persistent Storage:** Saves all tasks locally in a `tasks.json` file.

## 🛠️ Tech Stack

- **JavaScript / Node.js**
- Native `fs` (File System) module

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone this repository to your local machine:
