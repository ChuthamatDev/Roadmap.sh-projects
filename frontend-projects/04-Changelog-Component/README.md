# Changelog Component (Premium Edition)

A high-end, modern changelog component featuring a sleek dark theme, glassmorphism, and smooth animations. This project is part of my [Roadmap.sh](https://roadmap.sh/projects/changelog-component) frontend challenges.

## ✨ Premium Features
- **Modern Aesthetics**: Deep indigo/slate color palette with vibrant neon accents.
- **Glassmorphism**: Translucent surfaces with background blur for a sophisticated depth effect.
- **Dynamic Animations**: Smooth reveal animations for items and interactive glowing timeline dots.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Micro-interactions**: Hover effects that provide tactile feedback to the user.

## 🛠️ Technologies Used
- **Core**: Semantic HTML5 & Modern CSS3
- **Logic**: Vanilla JavaScript for dynamic content injection
- **Typography**: Google Fonts (Inter)
- **Icons**: SVG for crisp, scalable visuals

## 💻 Local Setup
1. Clone the repository.
2. Navigate to the `04-Changelog-Component` folder.
3. Open `index.html` in any modern browser.
